Text Formatting

By following the examples below, we will learn the basics of web page text formatting.

Headings (h1-h6)

Headings help define the structure and content hierarchy of your web page. They are defined with the h1 to h6 tags, where h1 is the most important heading, and h6 is the least important heading.

There are several theories on how to use h1-6 tags. The most important thing to note is that h1-h6 tags are not used to determine the size of the text. They serve to assign importance and value to your content. Content that is shown in capital letters is not necessarily an h tag.

It is extremely important to be consistent when using h1-h6 tags throughout the the website. Therefore, you should get to know the website and its content before you begin its realization.

Paragraph Usage

Paragraph (⟨p⟩) is one of the most basic tags for text formatting. It consists of the opening and closing tag, and it is a tag that you will use the most.

It represents the majority of the text and each individual entirety is written in a new paragraph.

Line Breaks

Sometimes, you will need to transfer to a new line(row), but stay within the same paragraph, meaning not create a new one. In order to that in HTML, you will need to use the line break tag (⟨br⟩).

Line break is an inline tag, which means you can use it within the h1-h6 tags or paragraph.

It is a self-closing tag.For example, if we have an address written in one paragraph (for now) LINE BREAK. Is what we can use to separate the entities within the address as new lines(rows).

Highlighting Text

Often times, we have the need to highlight certain parts of the text, (one word, a few words) or even a whole phrase without exiting the paragraph.

HTML gives you a few ways to do this.

-Bold Tag

Bold tag (⟨b⟩) is used when we want to highlight a certain part of the text (one word, a few words), or a whole phrase but ONLY VISUALLY. Upon consideration it is removed, and then reintroduced with HTML5.

In a visual sense in the browser it is presented by BOLDING the text.

-Strong Tag

Using the strong tag (⟨strong⟩) means that the part of the text that is between this tag represents the part of the text that has greater meaning form the rest of the text.

Visually, it represented as bolded text. However, this tag also gives importance to the text, as opposed to just the bold visual representation.

-Italic Tag

Just like the bold tag, we use the italic tag when we want to highlight a certain part of the text (one word, a few words) or a whole phrase but ONLY VISUALLY. There is no meaning to the text, it is only visually represented with slanted text in the browser.

-Emphasis Tag

Emphasis tag (⟨em⟩),"em" tag, is also used when we want to highlight a certain part of the text (one word, a few words) or a whole phrase. However, unlike the ITALIC TAG, that only serves as a visual representation the EMPHASIS TAG GIVES THE TEXT MORE MEANING AND IMPORTANCE, than the rest of the text.

In the browser it is visually represented with slanted text.

-UNDERLINE TAG

Underline Tag (⟨em⟩) is a tag that exclusively has VISUAL meaning only And it manifests by presenting the text between this tag like UNDERLINED TEXT
